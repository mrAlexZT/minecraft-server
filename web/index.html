Personal Minecraft Server

Your server is running Minecraft ver. VANILLA_VERSION



To start playing, launch you local Minecraft client and perform the following:


  1. Click on Change Profile button and ensure the Use version value corresponds to the Minecraft release your server is running (see it above).
    2. Save the changes and select the Play option.

  2. In the opened frame, choose Multiplayer.

  3. Then switch to either Direct Connect (for one time connection) or Add Server (to remember your choice) section and specify the ${ENDPOINT_URL} server link within the Server Address field.

    That's it! Now just Join Server you've specified and start your journey throughout Minecraft world.
